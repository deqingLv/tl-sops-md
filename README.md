# TL-SOPS-MD

> 技术 Leader 标准工作流程文档模板库

## 📖 项目简介

本项目旨在为技术 Leader 提供一套标准化的文档模板，通过 Markdown 格式的文档来承接和规范日常工作流程（SOP - Standard Operating Procedures），提升团队协作效率和知识沉淀质量。

## 🎯 核心思想

### 1. 文档化 SOP

技术 Leader 的标准工作流程可以通过文档来承接，将经验和方法论沉淀为可复用的模板，确保工作流程的标准化和一致性。

### 2. 模板化规范

通过文档模板来规范每个工作流程，确保：
- **一致性**：团队成员使用统一的格式和结构
- **完整性**：关键信息不会遗漏
- **可追溯性**：历史记录便于查阅和复盘

### 3. AI 友好的格式

选择 **Markdown** 格式的核心原因：
- ✅ **可被 AI Agent 阅读**：Markdown 是结构化的纯文本格式，AI 可以轻松理解和解析
- ✅ **提高工作效率**：AI 可以基于模板快速生成文档、提取信息、分析数据
- ✅ **跨平台兼容**：Markdown 被广泛支持，可在各种工具和平台中使用
- ✅ **版本控制友好**：纯文本格式便于 Git 等版本控制系统管理

## 📋 模板列表

### 技术类模板

- **故障复盘** (`01 技术/故障复盘.md`)
  - 参考 Google SRE 等大厂实践
  - 包含 Blameless 原则、5 Why 分析法
  - 完整的故障处理流程和复盘结构

- **专项治理** (`01 技术/专项治理.md`)
  - 技术问题专项治理全流程
  - 从问题识别到效果评估
  - 包含 RoadMap、风险控制、效果评估

- **最佳实践** (`01 技术/最佳实践.md`)
  - 技术最佳实践沉淀模板

### 管理类模板

- **会议纪要 - 正式版** (`03 管理/会议纪要-official.md`)
  - 完整的会议记录模板
  - 包含议程、讨论、决策、行动项等

- **会议纪要 - 简化版** (`03 管理/会议纪要-simple.md`)
  - 轻量级会议记录模板
  - 仅包含参会人、时间、会议要点

- **团队周报** (`03 管理/团队周报.md`)
  - 团队工作总结和计划模板
  - 包含完成事项、进行中事项、问题、计划等

- **团队成员周报** (`03 管理/团队成员周报.md`)
  - 个人周报模板

- **想法记录** (`03 管理/ideas.md`)
  - 想法和思路记录模板

## 🚀 使用方式

### 1. 直接使用模板

1. 找到对应的模板文件
2. 复制模板内容
3. 根据实际情况填写

### 2. 与 AI Agent 协作

由于模板采用 Markdown 格式，你可以：

- **让 AI 帮你填写**：提供上下文信息，AI 可以基于模板结构快速生成内容
- **让 AI 帮你分析**：将已填写的文档交给 AI，提取关键信息、生成摘要
- **让 AI 帮你优化**：让 AI 检查文档完整性、优化表达、补充遗漏


## 💡 最佳实践

1. **保持模板更新**：根据实际使用情况持续优化模板
2. **建立文档库**：将填写好的文档归档，形成知识库
3. **定期复盘**：回顾文档质量，提炼经验
4. **团队共享**：让团队成员了解和使用这些模板
